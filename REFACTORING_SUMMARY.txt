MCP TOOLS REGISTERED
====================

1. retrieve_context (RETRIEVAL)
   - Retrieves relevant documents using vector similarity
   - Parameters: query (str), k (int)
   - Returns: Documents with similarity scores

2. evaluate_confidence (EVALUATION)
   - Evaluates confidence in generated answers
   - Parameters: answer, hypotheses, retrieved_docs, quality_score
   - Returns: Comprehensive confidence metrics

3. store_documents (MEMORY)
   - Stores documents in vector store
   - Parameters: documents (list), metadata (list, optional)
   - Returns: Storage status and counts

4. score_retrieval_quality (EVALUATION)
   - Computes retrieval quality metrics
   - Parameters: retrieved_docs (list)
   - Returns: Retrieval metrics

5. compute_self_consistency (COMPUTATION)
   - Computes hypothesis agreement score
   - Parameters: hypotheses (list)
   - Returns: Self-consistency score

6. compute_evidence_diversity (COMPUTATION)
   - Computes diversity of evidence sources
   - Parameters: retrieved_docs (list)
   - Returns: Diversity score
